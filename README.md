## ü§î FAQs
### ‚ùì How to Scrape Google?

1. Open the `main.py` file.
2. Update the `queries` list with the locations you are interested in. For example:

```python
queries = [
  "Mango",
  "Watermelon",
]

Google.search(queries, max=10)
```

3. Run it.

```bash
python main.py
```

Then find your data in the `output` directory.
